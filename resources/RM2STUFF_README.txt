If you need rm2stuff repository:

Place it here as a submodule:
  git submodule add <RM2STUFF_URL> resources/rm2stuff

Or clone directly:
  cd resources
  git clone <RM2STUFF_URL> rm2stuff

Note: Submodules in resources/ directory:
  - rm2fb: remarkable2-framebuffer (for display compatibility)
  - rm2stuff: (to be added - please provide GitHub URL)
